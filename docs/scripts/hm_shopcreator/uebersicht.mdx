# hm_shopcreator

Erstelle und verwalte individuelle Shops direkt im Spiel — ohne Datenbankeinträge manuell zu pflegen.

## Features

- Shops in-game erstellen, bearbeiten und löschen
- Unbegrenzte Items und Kategorien pro Shop
- Kompatibel mit ox_inventory, qb-inventory und mehr
- Berechtigungssystem (Job, Gang, Ace-Permissions)
- Persistente Speicherung in der Datenbank

## Kompatibilität

| | |
|---|---|
| **Framework** | QBox, QBCore, ESX |
| **Inventar** | ox_inventory, qb-inventory, tgiann-inventory, qs-inventory |
| **Target** | ox_target, qb-target |
