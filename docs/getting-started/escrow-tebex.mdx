# Escrow & Tebex

Alle MopsScripts werden über das **CFX.re Asset Escrow**-System geschützt und vertrieben.

## Was ist Escrow?

Escrow ist ein Kopierschutzsystem von Cfx.re (FiveM). Teile des Scripts sind serverseitig verschlüsselt
und können nur auf lizenzierten FiveM-Servern ausgeführt werden.

<Info>
  Der Konfigurationsordner (`config/`) bleibt immer vollständig bearbeitbar.
</Info>

## Häufige Fragen

<AccordionGroup>
  <Accordion title="Mein Server startet das Script nicht">
    Stelle sicher, dass dein FiveM-Server-Account mit dem Tebex-Konto verknüpft ist,
    mit dem du das Script gekauft hast.
  </Accordion>
  <Accordion title="Kann ich den Code einsehen?">
    Der Konfigurationsbereich ist vollständig einsehbar. Der Core-Code ist aus Schutzgründen verschlüsselt.
  </Accordion>
  <Accordion title="Transfer auf einen anderen Server">
    Escrow-Lizenzen sind an deinen CFX.re-Account gebunden, nicht an einen spezifischen Server.
    Du kannst das Script auf beliebigen Servern deines Accounts nutzen.
  </Accordion>
</AccordionGroup>
