# Features

## Shop-Verwaltung

Shops werden vollständig in-game erstellt. Es ist kein Datenbankwissen erforderlich.

## NUI-Interface

Modernes UI im MopsScripts-Design — Farben sind in Colorpicker für jeden Individuell anpassbar, Plus Jakarta Sans, Bootstrap Icons.

## Berechtigungen

Shops können auf bestimmte Jobs, Gangs oder Ace-Permissions eingeschränkt werden.
