# Target-Systeme

MopsScripts unterstützen sowohl `ox_target` als auch `qb-target`. Das richtige System wird automatisch erkannt.

## Manuelle Auswahl

In der `config/config.lua` kannst du das Target-System manuell festlegen:

```lua
Config.Target = 'ox_target' -- 'ox_target' oder 'qb-target'
```
